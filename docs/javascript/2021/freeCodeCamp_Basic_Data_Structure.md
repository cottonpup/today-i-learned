# Basic Data Structure

## one-dimensional array vs multi-dimensional array

- one-dimensional array (중첩 배열을 담지 않는다)
- multi-dimensional array (복잡한 객체를 담을 수 있다)

## push() and unshift()

- push() 메서드는 엘리먼트를 배열의 마지막에 추가.

- unshift() 메서드는 엘리먼트를 배열의 앞에 추가.
